# LexicONE — функционал и технологии

## Краткое описание
LexicONE — веб‑приложение для изучения английских слов и фраз.  
Пользователь добавляет новые слова, повторяет их в режиме «Быстрое повторение» и получает подсказки (перевод, транскрипцию, примеры употребления).  
Данные могут храниться локально в браузере или синхронизироваться через базу данных с регистрацией.

## Основной функционал
- Добавление слов/фраз, автоперевод при вводе или ввод значения слова/фразы вручную.
- Быстрое повторение с алгоритмом интервального показа.
- Примеры употребления (ИИ‑генерация) и озвучка предложений.
- Озвучка слов/фраз.
- Проверка произношения (Google Cloud Speech) с оценкой по метрикам.
- Транскрипция (phonetic) для слов/фраз.
- Поиск по словарю, скрытие переводов, компактный/развернутый список.
- Авторизация и регистрация через Supabase.
- Статистика прогресса (кол-во слов, серия дней, дневная цель).
- Переключение языка интерфейса (RU/EN).

## Используемые технологии
- **Next.js 16 (App Router)** — фронтенд и API‑роуты.
- **React 19** — UI.
- **TypeScript** — типизация.
- **Tailwind CSS v4** — стили.
- **Supabase** — аутентификация и база данных.
- **Google Cloud Speech‑to‑Text** — фонетическая проверка произношения.
- **DeepSeek API** — генерация примеров употребления.
- **LibreTranslate / Google Translate (через прокси)** — автоперевод.
